# 🛒 MERN Ecommerce Website

A full-stack ecommerce application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). This project includes a customer-facing shopping website, an admin dashboard for managing products, and a backend API for handling authentication, products, and orders.

## 🚀 Features

### 👤 User Features
- User registration and login
- Browse products
- Search and filter products
- View product details
- Add products to cart
- Update cart quantity
- Place orders
- Secure authentication

### 🛠️ Admin Features
- Admin authentication
- Add new products
- Upload product images
- View product list
- Manage products
- Manage orders

### ⚙️ Backend Features
- REST API using Express.js
- MongoDB database integration
- JWT-based authentication
- Password encryption
- Image upload support
- Secure API routes
